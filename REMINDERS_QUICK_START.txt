FOLLOW-UP EMAIL REMINDERS - QUICK START
========================================

1. RUN MIGRATION:
   python migrate_add_owner_fields.py

2. SET UP EMAILJS:
   - Go to https://www.emailjs.com
   - Sign up (free account)
   - Create email service
   - Create email template (see EMAILJS_SETUP.md)
   - Get your Public Key, Service ID, Template ID

3. UPDATE DASHBOARD:
   - Open templates/dashboard.html
   - Find: emailjs.init('YOUR_PUBLIC_KEY_HERE');
   - Replace with your Public Key
   - Find: const EMAILJS_SERVICE_ID = 'YOUR_SERVICE_ID_HERE';
   - Replace with your Service ID
   - Find: const EMAILJS_TEMPLATE_ID = 'YOUR_TEMPLATE_ID_HERE';
   - Replace with your Template ID

4. TEST:
   - Create a Prospect deal
   - Set Owner Name and Owner Email
   - Set Next Follow-Up Date to TODAY
   - Go to Dashboard
   - Click "Check & Send Due Reminders"
   - Check your email!

REQUIRED FIELDS FOR PROSPECTS:
- Owner Name (required)
- Owner Email (required)
- Next Follow-Up Date (required, cannot be in past)

HOW IT WORKS:
- Checks for deals where Next Follow-Up Date <= today
- Only sends if not already sent today (prevents spam)
- Updates last_reminder_sent after successful send
- Shows banner if follow-ups are due
- Highlights overdue follow-ups in red
